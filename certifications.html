Advanced certification covering data pipeline architecture, ETL processes, database design, and big data technologies. Gained expertise in Python, SQL, NoSQL databases, Apache Spark, and data warehousing solutions.

Microsoft | 2024

Foundational understanding of core data concepts and Azure data services, including relational and non-relational databases, data analytics, and data processing workloads in the cloud.
