Certifications

Professional certifications in data engineering and analytics

AWS Certified Data Engineer - Associate

AWS| 2025

Associate-level certification validating skills in core data-related AWS services, ability to ingest and transform data, orchestrate data pipelines while applying programming concepts, design data models, manage data lifecycles, and ensure data quality.

Databricks Certified Data Engineer - Associate

Databricks| 2025

Professional certification demonstrating proficiency in the Databricks Data Intelligence Platform for data engineering tasks. Covers Apache Spark (SQL and PySpark) for ETL, Delta Lake for reliable data pipelines, workflow orchestration, data governance with Unity Catalog, and production deployment of data engineering solutions.
