Database Query Optimization Using Matrix Factorization

Duration: 6 months (ongoing) | Technologies: Python, NumPy, PostgreSQL, SQLite, Pandas, Scikit-learn

Developed a query optimization system that applies low-rank matrix approximations to accelerate database operations. Treated relational data as matrices and used Singular Value Decomposition (SVD) to approximate query operations, reducing storage complexity from O(nm) to O(k(n+m)) and computation from O(nm) to O(kn+km).

Built a query preprocessing layer that intercepts SQL queries, analyzes their structure using a cost model, and determines which operations benefit from matrix factorization. Implemented adaptive rank selection algorithm that automatically balances approximation quality against computational savings based on user-specified accuracy tolerance.

Impact/Results: Achieved 5.2× average query speedup on TPC-H benchmark (range: 2.1× to 12.8× depending on query structure), reduced memory usage by 3.8×, and maintained approximation error below 5% for 95% of queries. Greatest performance gains observed for aggregation-heavy queries and multi-table joins.

Fairness-Aware Recommendation System

Duration: 8 months (completed) | Technologies: Python, TensorFlow, PyTorch, NumPy, SciPy, Pandas, scikit-learn

Built a recommendation engine that explicitly balances prediction accuracy with fairness constraints using constrained optimization. Reformulated standard matrix factorization as multi-objective optimization problem with group fairness constraints (demographic parity), individual fairness constraints (similar users receive similar recommendations), and exposure fairness (minimum visibility for underrepresented items).

Developed three solution approaches: Lagrangian relaxation with fairness regularization, projected gradient descent onto constraint sets, and post-processing re-ranking. Implemented custom PyTorch layers for fairness-constrained gradient descent with real-time fairness metric computation and adaptive regularization parameter tuning.

Impact/Results: Reduced demographic parity difference from 0.24 to 0.08 (67% reduction) on MovieLens dataset while maintaining RMSE within 6% of baseline. In job recommendation experiments, increased exposure for underrepresented categories by 2.3× while maintaining precision@10 of 0.74 (vs 0.78 baseline). Reduced recommendation divergence between similar users by 54%.
